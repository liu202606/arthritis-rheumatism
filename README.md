This is the relevant code and derivative data for a study on arthritis and rheumatism. 
Among them, some environmental data were processed using ArcMap. 
If you need to use derivative data, please refer to the original source of the data:
[the National Centers for Environmental Information (NCEI) under the National Oceanic and Atmospheric Administration (NOAA)
(https://www.ncei.noaa.gov/access/metadata/landing-page/bin/iso?id=gov.noaa.ncdc:C00516); 
the China Vegetation Index Remote Sensing Inversion Dataset within the Resource and Environmental Science Data Registration and Publication System (Xu, 2018),
(https://www.resdc.cn/DOI/DOI.aspx?DOIID=49);
the National Tibetan Plateau/Third Pole Environment Data Center (Zhang, 2023):(https://data.tpdc.ac.cn/zh-hans/data/6854ebb3-8a60-454a-8d43-4e6a8c0ebd5d);(Peng, 2023)(https://www.tpdc.ac.cn/zh-hans/data/b7a5c363-3f63-4921-b657-8a4229476ec1).
